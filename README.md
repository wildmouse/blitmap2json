# blitmap2json

## How to use 

1. Create a `.env` file with `.env.example` file
1. Set a `INFURA_PROJECT_ID` to use client in `.env` (or replace client endpoint with you own)
1. Run a command with `tokenId` option with `-i` or `--tokenId`
